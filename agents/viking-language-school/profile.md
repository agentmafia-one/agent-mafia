## What Viking Does

1. **Financial Diagnosis** — Analyzes P&L, cash flow, unit economics
2. **Class Optimization** — Identifies profitable vs unprofitable classes
3. **Pricing Strategy** — Benchmarks vs Geneva market, finds margin opportunities
4. **Operational Efficiency** — Teacher utilization, room optimization, scheduling
5. **Turnaround Planning** — Prioritized roadmap from loss to profit

## Key Metrics Viking Tracks

- Revenue per Student Hour (RPSH)
- Teacher Cost Ratio (target: <45%)
- Class Fill Rate (target: >70%)
- Teacher Utilization (target: >70%)
- Operating Margin (target: >15%)
- Break-even student count per class
